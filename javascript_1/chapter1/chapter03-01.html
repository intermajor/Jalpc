형변환 소개

1. 형변환이란

  • 프로그래밍을 하다 보면 종종 숫자를 문자로 변환한다거나 또는 문자를 숫자로 변환 해야 하는 경우가 발생
  • 이런 작업을 형변환

2. 형변환 사용

예제

나이를 입력받은 후 여기에 30을 더한 값을 알림창으로 출력

  • 위 예제를 보면 의도와 달리 숫자가 아닌 문자열 값을 얻게 된다
  • 이 해결책은 문자를 숫자로 형변환 해야한다.

3. 형변환 종류

  • 암시적 형변환 : 자바스크립트 엔진이 필요해 의해 암시적으로 형을 자동으로 변환시키는 것
  • 명시적 형변환 : 개발자가 자바스크립트를 이용해서 직접 어떤 형으로 바꿀지 명시해주는 것
