초보자를 위한 함수

1. 함수란

  • 함수는 특정 기능을 하는 구문(알고리즘, 로직)을 묶어 재사용하는 문법. 일종의 포장방법

2. 함수 생김새

  • function이라는 키워드를 이용해서 만듬

예시

function 함수이름([매개변수1,매개변수2,...]){
    여러개의
    변수 + 연산자 + 형변환 + 조건문 + 반복문을
    활용해 만들어진 특정 기능을 하는 구문 (알고리즘)
    
    [return 값;]
}

3. 함수 호출

  • 함수이름 다음에 괄호를 열고 닫고 해주면 된다.
  • 함수이름 ();

4. 매개변수

  • 함수 외부에서 함수 내부로 값을 전달하는 방법 바로 매개변수 이용
  • 일종의 외부 데이터를 함수 내부로 전달하는 매개체 역할을 하는 변수

예시

함수A(데이터); 

function 함수A(매개변수1){
	매개변수1값 사용
	}

- 함수를 호출하면 데이터가 매개변수1로 넘어간다.
- 즉, 매개변수1=데이터와 같게 됨.

5. 리턴값

  • 함수 내부에서 함수 외보로 데이터를 보내기 위한 사용하는 통로
  • 함수 내부에서 함수 외보로 데이터를 보내기 위한 사용하는 통로

6. 함수 종류

  • 1. 자바스크립트 코어 함수(라이브러리)
  • 자바스크립트에서는 기본이 되는 기능을 미리 구현해 제공해준다, 기능을 하는 함수를 모아둔 것을 자바스크립트 코어 라이브러리라고 한다.
  • 2. 사용자 정의 함수
  • 말그대로 사용자가 필요로 해서 만드는 함수
