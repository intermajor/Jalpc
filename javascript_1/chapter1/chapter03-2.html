초보자를 위한 클래스

1. 클래스란

  • 연관 있는 변수와 함수를 하나로 묶을때 사용하는 문법

2. 클래스 사용

  • 실무 개발 하다보면 함수들이 많아진다. 이렇게 되면 관링하기 어려워진다.
  • 그래서 연관있는 요소들을 포장해 관리하는 문법을 제공하는데 이 기능이 바로 클래스

3. 인스턴스와 객체

  • 클래스를 사용하려면 일반적으로 인스턴스라는 것을 생성, 이때 new라는 키워드를 사용한다.
  • 다른 말로 객체라고 부른다.

예시

var tabMenu = new TabMenu();

4. 메서드와 프로퍼티

  • 클래스 내부에 만드는 변수를 프로퍼티 또는 멤버변수
  • 클래스 내부에 만드는 함수를 메서드 또는 멤버함수

5. 클래스 내부에 들어 있는 함수를 호출하는 방법

  • 인스턴트 만들기
  • new라는 키워드를 이용해 계산기 클래스의 인스턴스를 만든다.
  • 접근 연산자를 이용한 함수 호출
  • 접근 연산자를 이용해 원하는 기능인 함수에 접근해서 호출한다.

예시

var cal1 = new Calculator();
cal1.add(10,20);
