암시적 형변환

1. 암시적 형변환이란

  • 자바스크립트에 의해 자동으로 형변환이 일어나는 것

경우 결과 예
숫자 형+문자 형 문자 형 var a = 10+"10" // a는 문자 "1010"
불린 형+문자 형 문자 형 var a = true+"10" // a는 문자 "true10"
불린 형+숫자 형 숫자 형 var a = true+10 // a는 숫자 11

예제

[숫자를 문자로] 실행되면 변수 result에는 어떤 값이 저장될까요?


[불린 값을 숫자로] 실행되면 변수 result에는 어떤 값이 저장될까요?


[불린 값을 문자열로] 실행되면 변수 result에는 어떤 값이 저장될까요?
